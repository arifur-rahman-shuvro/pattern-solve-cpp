# FILE OVERVIEW

This file contain "LEFT ALIGNED TRIANGLE" pattern solve.

## Pattern overview

Have to print a left aligned triangle using starts(*).

## I/O example

Input: 5
Outout: *
        **
        ***
        ****
        *****

## Hint

- Use nested loop.
- Outer loop for row count
- Inner loop for print star in every rows
